JavaScript and Java

Despite their similar names, JavaScript and Java are quite different in design and functionality.

JavaScript is mainly used for client-side web development, while Java is a general-purpose language often used for server-side applications.

console.log("Hello World"); System.out.println("Hello World");

Loops

Loops are fundamental structures in programming that allow you to repeat a block of code as long as a specified condition is met.

You'll often find loops in tasks requiring iteration, such as traversing arrays or generating a series of numbers.

for (int i = 0; i < 10; i++) { }

Arrays

Arrays are data structures that can hold more than one value at a time.

They are particularly useful when you have a collection of similar types of data.

let arr = [1, 2, 3];
Functions

Functions are blocks of code designed to perform specific tasks.

They can take parameters, perform actions, and return the results of those actions.

function greet() { return "Hello"; }
